Compta-Free

www.compta-free.com

(Cliquez sur l'image pour voir la galerie)

Compta-Free est une application web pour P.M.E et cabinets d'expertise comptable.
Elle permet de visualiser en temps réel la trésorerie de l'entreprise grâce à l'API Bankin' qui récupère les données bancaires, mais aussi la gestion complète de son catalogue de services ainsi que de ses clients.
Côté cabinet d'expert comptable, elle facilite la réception des pièces comptables et le suivi des entreprises affiliées au cabinet. Ce projet m'a permis d'appréhender les technologies suivantes :

MongoDB
Création de la base de données en NoSQL, schématisation des données, problèmatiques de performances liées à une multitude de calculs et d'affichages de données bancaires.
React.JS
Création de composants React, ce qui facilite la maintenabilité de l'application. En effet React permet de diviser son travail en module ou composant réutilisable, qui évite la redondance dans le code et offre un gain de temps. Tout le front est en React.
PHP
Création d'un framework PHP "maison" style ModelController, notemment pour gérer les connections à la base de données ou encore pour le routing.
API REST
Création d'une API interne à l'application.
